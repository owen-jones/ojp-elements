Project dev can override any ojp-navigation-item css vars in their style sheet like this. See docs for full list of css vars.


There are only a handful of available variables to be overridden for the navigation item component. See examples of these styling differences in the example components listed below. 

            
  ojp-navigation-item {
    --ojp-navigation-item--vertical-link-padding: var(--space-xs); // Defaults to `var(--space-s)`
    --ojp-navigation-item--horizontal-link-padding: var(--space-xs) // Defaults to `var(--space-s)`
    --ojp-navigation-item--link-hover-color: $brand-blue; Defaults to `black`
  }

Example 1

This example shows how to change the padding around the nav items using spacer variables. It also shows how to change the hover-color of the links. 

              
  ojp-navigation-item {
    --ojp-navigation-item--vertical-link-padding: var(--space-m); // Defaults to `var(--space-s)`
    --ojp-navigation-item--horizontal-link-padding: var(--space-m) // Defaults to `var(--space-s)`
    --ojp-navigation-item--link-hover-color: red; Defaults to `black`
  }
