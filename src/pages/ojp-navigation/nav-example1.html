Usage

Example 1

This example shows how to change the background color and nav's position variable. In this example, the nav's position is set to relative and the background color is set to #eee. The nav will not follow down the page but will instead remain at the top of the page as you scroll. 

            
  ojp-navigation {
    --ojp-navigation--position: relative; // Default is `sticky`
    --ojp-navigation--background-color: #eee; // Default is `#ffffff`
  }
