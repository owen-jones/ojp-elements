Example 3

This example shows how changing a couple of key variables can create a "floating" effect for the navigation. The nav wrapper background color has been changed to transparent while the main-navigation here has a background color added to make it stand out. As you scroll down the page you will see the page content move behind the navigation and even peek over the top. The margin on the ojp-navigation__main-navbar has also been changed. This makes the additional space around the nav bar so that it is away from the edges of the screen. 

            
  ojp-navigation {
    --ojp-navigation--wrapper-background-color: transparent; // Default is the --ojp-navigation--background-color variable
    --ojp-navigation--main-navbar-margin: 10px 20px;  // Default is 0 0
  }
