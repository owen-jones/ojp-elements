Project dev can override any ojp-navigation css vars in their style sheet like this. See docs for full list of css vars


These are some of the variables that will be most frequently changed. Some may need to be overridden at a particular screen size (i.e. mobile or desktop) but not at another. See examples of these styling differences in the example components listed below. 

            
    ojp-navigation {
      --ojp-navigation--background-color: #eee; // Defaults to `#ffffff`
      --ojp-navigation--mobile-hamburger-line-color: red // Defaults to `#000000`
      --ojp-navigation--menu-height: 80vh; Defaults to `100vh` at mobile and `auto` at desktop
      --ojp-navigation--menu-width: 100%; Defaults to `100vw` at mobile and `auto` at desktop
    }
