Overview

See auto-generated component docs here


Project dev can override any ojp-footer css vars in their style sheet like this. See docs for full list of css vars:

          
        // Default values
        ojp-footer {
          --ojp-footer--background-color: #000;
          --ojp-footer--color: #fff;
          --ojp-footer--padding: 0.25em;
          footerlogo = 'https://via.placeholder.com/150';
        }

Usage

Example 1

This is a simple footer with custom text

          
            <ojp-footer
              id="footer-1"
              class="ojp-footer"
              footertext="© 2024 Owen Jones Partners"
            ></ojp-footer>

Example 2

This is a centered footer with default text

        
          <ojp-footer
            id="footer-2"
            class="ojp-footer"
            iscentered
          ></ojp-footer>

Example 3

This is a centered footer with a custom background color and custom text and custom text color

        
          <ojp-footer
            id="footer-3"
            class="ojp-footer"
            footertext="© 2024 Owen Jones Partners"
            footerbackgroundcolor="hotpink"
            footertextcolor="white"
            iscentered
          ></ojp-footer>

Example 4

This is a centered footer with a custom background color and custom text and custom padding

        
          <ojp-footer
            id="footer-4"
            class="ojp-footer"
            footertext="© 2024 Owen Jones AND Partners"
            footerbackgroundcolor="violet"
            iscentered
            footerpadding="20px"
          ></ojp-footer>

Example 5

This is a non centered footer with a custom background color and custom text and custom padding, with custom LOGO

        
          <ojp-footer
            id="footer-5"
            class="ojp-footer"
            footertext="© 2024 Owen Jones AND Partners"
            footerbackgroundcolor="violet"
            footerpadding="10px"
            footerlogodisplay="true"
            footerlogo="https://whoisowenjones.com/assets/show-dont-tell__text--mobile.svg"
          ></ojp-footer>

Example 6

This is a centered footer with a custom background color + text color, and custom text and custom padding, with default LOGO

        
          <ojp-footer
            id="footer-6"
            class="ojp-footer"
            footertext="© 2024 Owen Jones AND Partners"
            footerbackgroundcolor="darkviolet"
            footerpadding="10px"
            footerlogodisplay="true"
            footertextcolor="white"
            iscentered
          showSocialIcons=true
    socialIconsList='[{"icon": "https://example.com/icon1.png", "link": "https://example.com/profile1"}, {"icon": "https://example.com/icon2.png", "link": "https://example.com/profile2"}]'

          ></ojp-footer>
